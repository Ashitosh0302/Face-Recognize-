# Face Recognition System using PCA (Eigenfaces) and ANN

## üìå Project Overview

This project implements a **Face Recognition System** using **Principal Component Analysis (PCA)**, also known as the **Eigenfaces method**, combined with an **Artificial Neural Network (ANN)** for classification.

The system performs face image preprocessing, dimensionality reduction, classification, performance evaluation, and **impostor rejection testing** using prediction confidence thresholds.

---

## üéØ Objectives

- To recognize human faces using machine learning techniques  
- To reduce high-dimensional image data using PCA (Eigenfaces)  
- To classify faces using an Artificial Neural Network  
- To analyze system performance for different PCA component values  
- To detect and reject impostor faces

## üß† Methodology

### 1. Image Preprocessing
- Images are loaded from the dataset
- Converted to grayscale
- Resized to **100 √ó 100 pixels**
- Normalized to range **[0, 1]**
- Flattened into vectors

---

### 2. Feature Extraction (PCA ‚Äì Eigenfaces)
- PCA is applied to reduce dimensionality
- Important facial features are extracted as **eigenfaces**
- Whitening is applied for better classification
- Different numbers of PCA components (k values) are tested

---

### 3. Classification (ANN)
- A **Multi-Layer Perceptron (MLP)** is used
- Two hidden layers: **100 ‚Üí 50 neurons**
- ReLU activation function
- Adam optimizer
- Backpropagation for training

---

### 4. Performance Evaluation
- Training and testing accuracy
- Accuracy vs number of PCA components
- Visual inspection of predictions

---

### 5. Impostor Testing
- Synthetic impostor faces generated by adding noise
- Prediction confidence evaluated
- Faces below confidence threshold are rejected
- Impostor rejection rate calculated
